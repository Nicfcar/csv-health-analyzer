# csv-health-analyzer
programa simples que leia um arquivo CSV com dados de pacientes (como idade e pressão arterial) e calcule médias e estatísticas básicas.
Para utilizar o programa coloque o nome da onde o arquivo se encontra. No exemplo que esta ai utilize python3 main.py pacientes.csv


<h1> o arquivo csv disponivel é do data set de diabetes disponivel no site da Kaggle </h1>
